Репозиторий для работы с Python и хранения в нём созданных скриптов.
Для работы проекта используются:
 Python версии 3.14.6,
 Git версии 2.53.0.windows.2.

1. На GitHab создан основной репозиторий "PY_homework"
2. Создан локальный учебный репозиторий "test-repo-py" для настройки окружения и отработки тестовых заданий. 
3. Для выполнения заданий первого урока "Переменные, списки и функции Python" создана папка lesson_01. В ней находятся файлы с выполненными скрпитами по заданиям:
- script-traning.py - тренировочные упражнения урока
- lesson_1_task_1.py - домашнее задание "Создание переменной"
- lesson_1_task_2.py - домашнее задание "Перезапись переменной"
- lesson_1_task_3.py - домашнее задание "Получение пользовательского ввода"
- lesson_1_task_4.py - домашнее задание "Создание функции"
- lesson_1_task_5.py - домашнее задание "Параметризация функций"
